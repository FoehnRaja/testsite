PROJECT ToDo

Project ToDo was a final project requirement our Professor gave us during our Application Development subject. The application is created in React Native with the help of an open source platform, Snack expo.

the android application's sole purpose is to have an application that can create a list of tasks and delete them as you complete them. A very simple to do list application.
